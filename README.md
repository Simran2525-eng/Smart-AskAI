# Smart-AskAI
Beginner AI agent that answers questions and fetches real-time info via a simple chat interface.

Basic AI Agent

This is a beginner-level AI agent created for learning and experimentation with artificial intelligence and real-time information.
What it does

Answers questions:You can ask the agent general questions and it provides simple, meaningful answers.

Fetches real-time information: For example, it can tell the current weather in any city or search for nearby places like cafes or restaurants.

Chat interface: The agent works like a simple chat, making it easy to interact with.

Purpose
This project is designed to help beginners understand:
How AI agents work
How real-time information can be used in AI
How to interact with AI through simple prompts
It is a learning project meant for experimentation and understanding the basics of AI.

Status
Beginner-level project
Functional but limited in features
Improvements and new ideas are planned for the future

Security Note
No API keys or sensitive data are included in this project
Users would need their own API keys to fetch real-time data
